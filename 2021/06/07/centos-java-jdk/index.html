Centos7.X安装Jdk

June 07, 2021 689

JDK是 Java 语言的软件开发工具包，主要用于移动设备、嵌入式设备上的java应用程序。JDK是整个java开发的核心，它包含了JAVA的运行环境（JVM+Java系统类库）和JAVA工具。

JDK下载

  • 由于下载速度原因，此处jdk采用华为镜像（https://repo.huaweicloud.com/java/jdk/）

JDK解压

  • 将下载好的JDK保存在自己所创建的路径，并通过tar命令解压。

配置JDK

  • 使用 vi /etc/profile 命令，编辑 /etc/profile文件

生效JDK

  • 使用 source /etc/profile 命令，生效JDK的配置

查看JDK

  • 使用 java -version 命令，测试JDK的配置是否成功
